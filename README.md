# Notes App (Vanilla JavaScript)

A simple Notes application built with HTML, CSS, and JavaScript.

## Live Demo
https://den766.github.io/notes-app-vanilla-js/

## Features
- Add notes
- Input validation
- Dynamic DOM rendering
- Search Notes
- Delete notes
- Edit notes
- Pin & unpin Notes
- Local Storage Persistance

## Tech Stack
- HTML
- CSS
- JavaScript

## How It Works
Notes are stored in the browser using LocalStorage.  
The UI dynamically updates whenever notes are created, edited, or deleted.

This project is part of strengthening my JavaScript fundamentals before moving deeper into the MERN stack.
